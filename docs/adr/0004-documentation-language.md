# 0004: 문서 언어 결정

## 문제 상황

문서를 작성할 때 쓸 언어를 결정해야 한다.

## 결정

### 고려했던 대안

#### 영어

- **장점**: 세계 공용어로서 다양한 사람들이 이해할 수 있음
- **단점**: 영어가 모국어가 아니므로 글 작성에 어려움이 있음

#### 한국어

- **장점**: 적기 편함
- **단점**: 한국인이 아닌 경우 일반적으로 이해하기 힘듬

### 최종 결정

한국어를 사용하도록 한다.

본질적으로 이 프로젝트의 목적은 훌륭한 품질의 한국어 자료의 양을 늘리는 데에 있다.
따라서, 한국어를 주로 사용하도록 한다.

## 결과

아키텍처 결정 기록을 한국어로 작성한다.
